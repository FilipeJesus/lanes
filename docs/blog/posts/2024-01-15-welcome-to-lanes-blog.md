# Welcome to the Lanes Blog!

We're excited to launch the official Lanes blog. This space will serve as a hub for sharing projects built with Lanes, tutorials to help you get the most out of parallel AI coding, and updates about new features.

## What to Expect

### Project Showcases
See what others are building with Lanes. From CLI tools to web applications, discover real-world examples of parallel AI workflows in action.

### Tutorials & Guides
Step-by-step guides covering everything from basic setup to advanced workflow configurations. Learn how to leverage Lanes' structured workflows for reliable, repeatable results.

### Feature Announcements
Stay up-to-date with the latest improvements to Lanes, including new workflow templates, agent specializations, and integration options.

## Share Your Projects

Have you built something cool with Lanes? We'd love to feature it! Submit your project for consideration by opening an issue on our GitHub repository with the "blog" label.

Happy coding!
